# Portfolio Website

This repository contains a fully designed personal portfolio website built with HTML, CSS, JavaScript, and Bootstrap. It includes a responsive homepage with multiple sections, dedicated pages for projects and blogs, custom styling, scripted UI behavior, and a complete set of images and assets.

The site is suitable for presenting personal information, showcasing services and work, sharing blog content, and linking to social media profiles.

---

## Overview

The website provides:

* A responsive navigation bar with Bootstrap
* Smooth scrolling and UI behavior scripting
* A professional home section
* About section with profile content
* Services section with multiple highlighted capabilities
* Portfolio/gallery section with images
* Contact area with social links and icons
* Separate **Projects** page
* Separate **Blog** page
* Custom styling and JavaScript functionality

---

## Features

* Clean, modern portfolio layout
* Bootstrap 5 responsive design
* Custom CSS styling
* Font Awesome icons
* Navbar color behavior on scroll
* Multiple ready-to-use sections:

  * About
  * Services (including Website Development, Website Design, AI Developer, SEO, DevOps, QA)
  * Portfolio
  * Contact
* Social media links (GitHub, LinkedIn, Instagram)
* Separate Project Portfolio page with categorized project cards

* Dedicated images and assets

## Installation and Setup

1. Download or clone the repository.
2. Extract the folder if downloaded as a ZIP.
3. Open `index.html` in any modern web browser.

No additional setup, server, or build steps are required.

---

## Usage

* Open `index.html` to view the main portfolio site.
* Navigate to:

  * `Project.html` to view the project portfolio page.
  * `Blog.html` to view the blog page.
* Modify HTML, CSS, JavaScript, and image files as needed to personalize content.

---

## Notes

* All external libraries such as Bootstrap and Font Awesome are loaded via CDN.
* JavaScript files handle navbar scrolling effects and UI behavior.
* Images are stored inside the `/images` directory.
* Custom styles are located in the `/style` directory.
* Additional interactive behavior scripts are inside `/script`.
